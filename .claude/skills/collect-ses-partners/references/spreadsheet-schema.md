# スプレッドシート列定義

## シート1（本番）

| 列 | 項目名 | 説明 |
|---|---|---|
| A | 企業名 | SES企業の正式名称 |
| B | フォーム送信 | チェックボックス。ヘッダーにCOUNTIF等で送信済み数を表示 |
| C | URL | 企業サイトのURL |
| D | お問い合わせURL | お問い合わせフォームのURL |
| E | 面談について | A/B等のランク（手動入力） |
| F | 概要 | 企業概要テキスト（手動入力） |
| G | メモ | 備考（手動入力） |
| H | 正規化企業名 | 重複チェック用ヘルパー列（Spreadsheet関数） |
| I | 正規化URL | 重複チェック用ヘルパー列（Spreadsheet関数） |

- B列ヘッダー: `=COUNTIF(B2:B,"TRUE")&"/"&COUNTA(B2:B)&" 送信済"` 等で自動集計
- 新規追加時: E〜G列は空欄、B列はFALSE

## 作業用シート

| 列 | 項目名 | 説明 |
|---|---|---|
| A | 企業名 | スクレイピングで取得した企業名 |
| B | フォーム送信 | FALSE（固定値で初期化） |
| C | URL | 企業サイトURL |
| D | お問い合わせURL | お問い合わせフォームURL |
| E | URL取得状態 | `URL済` / `取得失敗` / `アクセスエラー` |
| F | ソース | `new` またはポータルサイト名 |
| G | 正規化企業名 | 重複チェック用ヘルパー列（Spreadsheet関数） |
| H | 重複フラグ | TRUE=重複（統合除外）/ FALSE=新規（統合対象） |

## ポータル管理シート

| 列 | 項目名 | 説明 |
|---|---|---|
| A | ポータルURL | ポータルサイトのURL |
| B | ポータル名 | サイト名 |
| C | 最終クロール日 | 最後にクロールした日付（YYYY-MM-DD） |
| D | 取得企業数 | そのポータルから取得した企業数 |
| E | ステータス | `クロール済み` / `エラー` |

## 重複判定ルール

### 正規化ルール

- 企業名: 「株式会社」「（株）」「(株)」除去、全角半角スペース除去、英字小文字化
- URL: `https://` / `http://` 除去、`www.` 除去、末尾スラッシュ除去

### Spreadsheet関数

G列（正規化企業名）:
```
=SUBSTITUTE(SUBSTITUTE(SUBSTITUTE(SUBSTITUTE(A2,"株式会社",""),"（株）",""),"(株)","")," ","")
```

I列（正規化URL）:
```
=REGEXREPLACE(REGEXREPLACE(REGEXREPLACE(C2,"^https?://",""),"^www\.",""),"/$","")
```

H列（重複フラグ）:
```
=OR(
  SUMPRODUCT(EXACT(INDIRECT("シート1!H2:H"),G2))>0,
  SUMPRODUCT(EXACT(INDIRECT("シート1!I2:I"),I2))>0
)
```

### 判定結果

- TRUE = 重複（シート1統合除外）
- FALSE = 新規（シート1統合対象）
